= Vert.x Maven Starter

This project is a template to start your own Vert.x project using Apache Maven.

== Prerequisites

* Apache Maven
* JDK 8+

== Getting started

[source]
== build the project

To run the project, just use:
----
mvn clean install 
----
this will create a fat jar in target

== run the project

To run the project, just use:

----
mvn clean compile exec:exec@ex
----
